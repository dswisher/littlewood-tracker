# LittleWood Tracker

A little dotnet app to track your progress in the game LittleWood by reading a save file.


## Inspiration

- [luisdcb97/LittleTownViewer/](https://github.com/luisdcb97/LittleTownViewer/) - Java - A save editor for the game Littlewood
- [ClearwaterUK/LittleEdit](https://github.com/ClearwaterUK/LittleEdit) - C++ - A save editor for Littlewood. No longer maintained.
